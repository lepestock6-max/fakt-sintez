# Бот для озвучивания ваших интересных фактов пробуйте!

#Как запустить бота

```
py -m venv ven
```

```
Set-ExecutionPolicy RemoteSigned -Scope Process
```

```
.\venv\Scripts\Activate.ps1
```

```
1 - Пиши /start чтобы запустить бота.
```

```
2 - Напишите /fact чтобы бот написал и озвучил вам интересный факт!
```
